# Ordbyggaren

Phonological training app — build words from sounds and letters.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install ordbyggaren
```

### Fedora/RPM
```bash
sudo dnf install ordbyggaren
```
